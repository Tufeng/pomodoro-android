# Proposal: 时间滑块磁吸锚点

**Change ID**: add-timer-slider-snap  
**状态**: 待审批

## Why（为什么做）

当前时间设置使用连续滑块，用户难以精确停在常用时间值（如 25、30、45、60 分钟）：
- 手指精度不足，容易停在 26 或 24 分钟等非整数
- 增加设置摩擦，与「轻松配置节律」的产品理念相悖

## What Changes（做什么）

1. 在滑块上为 25、30、45、60 分钟设置可见锚点：
   - 锚点显示为刻度线 + 数字标注
   - 锚点颜色与普通刻度区分（主题色高亮）
2. 滑块拖动逻辑：当滑块落点距任意锚点 ≤ 2分钟时，自动吸附到该锚点
3. 吸附时触发轻微触感反馈（HapticFeedback.createOneShot）

## Impact（影响）

- **组件**：现有 Slider 组件替换为自定义 `SnapSlider` 组件
- **范围**：工作时长、短休时长、长休时长三个滑块均应用
- **权限**：无需新增权限，HapticFeedback 使用系统 Vibrator 服务
